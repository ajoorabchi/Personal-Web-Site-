Title: State-space Planning by Integer Optimization

Author:

Author:

Subject: Henry Kautz,Joachim P. Walser State-space Planning by Integer Optimization

Description: This paper describes ILP-PLAN, a framework for solving AI planning problems represented as integer linear programs. ILP-PLAN extends the planning as satisfiability framework to handle plans with resources, action costs, and complex objective functions. We show that challenging planning problems can be effectively solved using both traditional branchand -bound IP solvers and efficient new integer local search algorithms. ILP-PLAN can find better quality solutions for a set of hard benchmark logistics planning problems than had been found by any earlier system. 1 Introduction In recent years the AI community witnessed the unexpected success of satisfiability testing as a method for solving state-space planning problems (Weld 1999).
