Description: of the Thesis Parallel Simulation of Parallel I/O Programs by Andy Chih-Yuan Kahn Master of Science in Computer Science University of California, Los Angeles, 1997 Professor Rajive Bagrodia, Chair Recent trends in computing have shown vast improvements in processor speed. Although this allows computers to solve ever larger problems, the input/output (I/O) subsystem has always been orders of magnitudes slower than the processor. Relative to processor speed, I/O has not shown as much improvement. This results in I/O performance as a major bottleneck. Just as using more processors in parallel can potentially lead to faster processing, I/O can also be done in parallel. Parallel disk subsystems, parallel file systems, and algorithms for collective I/O have been proposed. However, there are many tradeoffs when using these components for parallel I/O.
