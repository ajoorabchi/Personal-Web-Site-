Title: Conflict-Aware Replication for Dynamic Content Web Sites

Subject: Conflict-Aware Replication for Dynamic Content Web Sites

Description: Conict-aware replication is a novel lazy replication technique for scaling the back-end database of a dynamic content web server using a cluster of commodity computers. This technique provides both throughput scaling and 1-copy serializability. It has generally been believed that this combination is hard to achieve through replication because of the growth of the number of conicts.
