Subject: Michael Heinrichs,Name Michael Heinrichs,Dr. Hausi,A. Muller Evaluating a Focus+Context Zoom Interface in Complement with Hypertext as a Program Understanding Tool

Description: Computer programmers are faced with the task of building and maintaining increasingly complex software systems. Since program understanding makes up a large part of program maintenance, tools supporting program understanding should increase the efficiency of program maintenance. This thesis presents a prototype tool to help visualize and navigate program source code. Many traditional visualizations of software structure require a great deal of screen real estate, preventing the programmer from viewing the entire structure at once, and increasing the difficulty of navigating and understanding the information space. In addition, specialized software displays can pose a barrier to tool adoption when they require additional training, and show poor integration with existing tools. The prototype tool (JavaZoom) uses the Continuous Zoom interface to allow focus+context displays of the complete software structure.
