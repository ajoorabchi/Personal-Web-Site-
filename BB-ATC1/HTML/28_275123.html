Title: Integrating State Charts in Specware and Aspects of Correct Oberon Code Generation

Subject: Integrating State Charts in Specware and Aspects of Correct Oberon Code Generation

Description: State charts are finite state machines with hierarchical structuring and several models of communication. They are widely used in hardware, software, and communication industries to specify reactive and concurrent systems. In absence of a common semantics for state charts, we use Evolving Algebras, that showed to be powerful enough to model all types of sequential, parallel, distributed and real-time applications of dynamic systems. The generality of evolving algebras allows to mimic all state chart dialects. We use a many-sorted higher-order logic algebraic specification language with loose semantics to describe the states of evolving algebras. We have found that the hierarchical structuring operations of state charts coincide with the basic composition operations of the used specification language SLANG.
