Title: Parallel Discrete-Event Simulation Using Pvm

Author:

Author:

Subject: Wentong Cai,Hung Khoon Tan Parallel Discrete-Event Simulation Using Pvm

Description: In this paper, we investigate the performance of parallel discreteevent simulation using PVM, and the feasibility of using PVM for object-oriented parallel program development. We find that PVM can be very easily adapted by C programmers for parallel programming on a network of workstations, but the performance of the program tends to depend on the workload of each workstation. In our study, both deadlock avoidance and time-warp approaches are used for time synchronization in parallel simulation, and the performance analysis are conducted on simulating Omega networks and public telephone networks.The design and implementation of our simulation program are described. The results obtained are also presented. Keywords: parallel discrete-event simulation, deadlock avoidance, time-warp, PVM, Omega network & telephone network simluation, object-oriented programming. 1.
