Title: Scalability Analysis of Parallel Finite Element Methods using Performance Simulation

Author:

Subject: A. J. C. Van Gemund Scalability Analysis of Parallel Finite Element Methods using Performance Simulation

Description: this paper we present a performance simulation approach to the performance prediction of macro data flow applications on distributed-memory machines. The prediction approach is used to evaluate the effect of mapping strategies and machine parameters on the performance of FEM solvers on a transputer mesh. A considerable improvement in simulation speed is achieved through model reduction. At the same time, results show that the model is accurate within a few percents.
