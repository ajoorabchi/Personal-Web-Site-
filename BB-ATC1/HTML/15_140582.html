Title: DEFLATE Compressed Data Format Specification version 1.3

Subject: DEFLATE Compressed Data Format Specification version 1.3

Description: This specification defines a lossless compressed data format that compresses data using a combination of the LZ77 algorithm and Huffman coding, with efficiency comparable to the best currently available generalpurpose compression methods. The data can be produced or consumed, even for an arbitrarily long sequentially presented input data stream, usingonly an a priori boundedamount of intermediate storage. The format can be implemented readily in a manner not covered by patents.
