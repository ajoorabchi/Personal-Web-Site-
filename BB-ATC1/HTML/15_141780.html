Title: A Flexible Protocol Architecture for Multi-party Conferencing

Author:

Author:

Subject: Sudhir Aggarwal,Sanjoy Paul A Flexible Protocol Architecture for Multi-party Conferencing

Description: This paper proposes a new multicast protocol architecture that can be used for conferencing or online discussion groups. Sessions of this type have the characteristic that while all group members are usually listeners, several members can and often do become speakers during the course of the session. Another characteristic is that the group membership can change as members drop out or are added to the session. Thus, one major objective is to design a protocol that is effective when there are many such sessions that are concurrently active. A second objective is to use the same basic architecture to implement multicasting in a mobile environment. This is possible because the basic primitives for joining and leaving discussion groups are similar to what is needed for mobile applications that need to be connected through various mobile support stations.
