Title: Software Support for Parallel Structured Adaptive Mesh Methods

Subject: Scott R. Kohn,Stephen J. Fink,Scott B. Baden Software Support for Parallel Structured Adaptive Mesh Methods

Description: Introduction Adaptive mesh refinement plays an important role in resolving physically localized phenomena in a variety of scientific and engineering disciplines. However, adaptive mesh refinement implementations require appropriate run time support to manage their underlying complexity, especially on parallel computers. We have developed a software support infrastructure for adaptive mesh methods running on parallel architectures. This infrastructure has been applied to various applications, including electronic structure calculations [6] and spatially extended dynamical systems [9]. At the heart of the infrastructure is the KeLP system, a C++ class library offering portable performance across a wide range of parallel architectures. KeLP is based on LPARX [8], and it provides the flexibility needed to effectively manage non-uniformly decomposed data at run time, along with the underlying irregular communication patterns.
