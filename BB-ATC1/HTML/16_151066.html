Title: Future Work

Subject: Future Work

Description: Classes At present the Testgraph Framework does not provide a direct mechanism for the testing of abstract classes. It is possible to test an abstract class in a similar manner to that used by the ACE tool. However, this is not automated as in the ACE tool and thus involves considerable programming effort on the part of the tester in order to Page 49 produce, record and verify the output messages. Automated support for the testing of abstract classes would substantially reduce the burden on the tester and reduce the likelihood of errors entering the test suite. Scaling-Up Problems Before any new technology can be used with confidence in industry we must be certain that it can be applied successfully to large scale projects. Various work in the literature on testgraphs [15,16] and in this report goes some way towards addressing the scaling-up problems mentioned in Chapter 3.
