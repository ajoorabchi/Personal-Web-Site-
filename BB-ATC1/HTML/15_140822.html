Description: Introduction The problem of learning the statistical properties of the environment has already received some attention 1 Gamma 7 (See also Ref. 8 for a review). For instance, for a noiseless channel the generalized Oja rule 3 produces synaptic couplings between an input and an output layer with M neurons that converge to values that span the same subspace as the M principal components. For a similar problem but including the effect of noise Linsker 5 has considered the infomax principle and, under some conditions, he exhibited a solution for the optimal couplings. This work however leaves many questions open about the behaviour of the network under different or more general conditions.
