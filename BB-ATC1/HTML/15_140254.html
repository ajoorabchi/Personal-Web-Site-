Title: A Self-Organizing Map that Learns the Semantic Similarity of Reusable Software Components

Subject: A Self-Organizing Map that Learns the Semantic Similarity of Reusable Software Components

Description: . This paper is concerned with the application of Kohonen's self-organizing map in the area of software reuse. Although software reuse has a long historical tradition in research, what is still missing is an appropriate way to structure software libraries according to the semantic similarities of the stored software components. In this paper we describe an approach to overcome this inconvenience by applying Kohonen's selforganizing map to the description of software components. As a result we obtain a semantically structured software library which is paramount to conventional approaches. 1. Introduction The area of software reuse is concerned with the development of software systems by using already existing components instead of developing the whole system from scratch. In general, reusable entities during the software development process are the following: object-code modules, source-code modules, specifications, and test data.
