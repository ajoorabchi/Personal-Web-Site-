Title: Facilitating Open Communication in Agent Systems: the InfoSleuth Infrastructure

Author:

Subject: M. H. Nodine Facilitating Open Communication in Agent Systems: the InfoSleuth Infrastructure

Description: . This paper addresses issues in developing multiagent systems --- those in which it is easy to expand the functionality by adding new agents with new capabilities, and those which facilitate interoperability with other agent systems. In this paper, we argue that an open multiagent system should define the following support elements for agent communication: 1. A common set of speech acts to define the types of messages that an agent might send to another agent. 2. A common service ontology by which the agents can describe their capabilities to each other, and reason about which agents have the capabilities needed to execute specific tasks. 3. A common set of prescriptive conversation policies to define the acceptable exchanges of messages between agents.
