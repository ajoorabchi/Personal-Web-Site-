Title: Implementation and Evaluation of Nested Task and Data Parallelism for High Performance Fortran within the ADAPTOR Compilation System

Author:

Subject: Thomas Brandes Implementation and Evaluation of Nested Task and Data Parallelism for High Performance Fortran within the ADAPTOR Compilation System

Description: Task parallelism has been proven to be useful for applications like real-time signal processing, branch and bound problems, and multidisciplinary applications. The new standard HPF 2.0 of the data parallel language High Performance Fortran (HPF) provides approved extensions for task parallelism that allow nested task and data parallelism. Unfortunately, these extensions allow the spawning of task but do not allow interaction like synchronization and communication between tasks during their execution and therefore might be too restrictive for certain application classes. E.g., they are not suitable for expressing the complex interactions among asynchronous tasks as required by multidisciplinary applications. Widely accepted parallel programming paradigms like farming are not expressible. This paper describes the support of task parallelism in High Performance Fortran and its realization within the ADAPTOR HPF compilation system.
